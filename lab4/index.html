Lab 4: Pseudocoding and Problem-solving

Challlenge

In this lab, we had to understand how to pseudo code in preperation for JavaScript programing. This will require us to breakdown an everyday task into organized steps and to add comments for a game of rock paper scissors as if we were planning to code it in the future.

Problems

The main problem we faced was condensing everday tasks into a very robotic and organized list. This also goes for the pseudo coding of the Rock,Paper, Scissors game, since we had to pretend we were planning on coding a game with JavaScript.

Results

Everyday Task

Charging an electric bike:

    1. Bring bike into dorm
    2. Pull out charging cord
    3. Place bike next to an outlet
    4. Open charging input port cover
    5. Plug charger input into input port
    6. Plug lightning into the outlet
    7. Wait 4 to 6 hours for the bike to fully charge
    8. Pull out the cord when fully charged
    9. Ride off into the sunset

Pseudocode a computer task

Rock, Paper, Scissors:

    1. Layout out the 3 options
    2. Accept user input
    3. Randomize computer input
    4. Evaluate results
    5. Test for win condition
